# ShopifyNexus

A hands-on project integrating Shopify Headless Commerce with Next.js App Router to get familiar with real-world e-commerce business requirements.

## Description
ShopifyNexus acts as an independent frontend (Headless) that connects to a Shopify store via the GraphQL API. The project simulates core e-commerce functionalities ranging from browsing products and managing the cart to handling data synchronization via Webhooks.

## Purpose
*   Explore and get familiar with the Shopify ecosystem (Shopify Admin, Storefront API).
*   Practice API query design and implementation using GraphQL.
*   Consolidate and apply modern Next.js features (App Router, Server Components, Caching, ISR).

## Tech Stack
*   Framework: Next.js
*   Styling: Tailwind CSS v4
*   API Protocol: GraphQL (Shopify Storefront API)
*   Language: TypeScript

## Goals
*   Build product listing and detail pages optimized for performance and SEO.
*   Integrate shopping cart functionality and session synchronization using Shopify Cart API.
*   Develop flexible product filtering and real-time search.
*   Establish API routes to listen to Webhooks from Shopify Admin.

## Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend-next
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the project locally:
   ```bash
   npm run dev
   ```
